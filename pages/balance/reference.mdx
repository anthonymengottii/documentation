## Saldo

Consulte seu saldo atual e movimentações recentes.

<Note>
  O saldo é calculado automaticamente com base nas transações e movimentações financeiras.
</Note>

## Endpoints Disponíveis

- **GET** `/api/v1/balance` - [Obter saldo](/pages/balance/get)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
