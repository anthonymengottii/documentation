## Produtos

Gerencie seu catálogo de produtos que podem ser associados a links de pagamento.

<Note>
  Para um guia completo sobre Produtos, consulte [Products Guide](/pages/guides/features/products).
</Note>

## Endpoints Disponíveis

- **POST** `/api/v1/products` - [Criar um novo produto](/api-reference/api/v1/products/post)
- **GET** `/api/v1/products` - [Listar produtos](/api-reference/api/v1/products/get)
- **GET** `/api/v1/products/{id}` - [Obter produto específico](/api-reference/api/v1/products/{id}/get)
- **PATCH** `/api/v1/products/{id}` - [Atualizar produto](/api-reference/api/v1/products/{id}/patch)
- **DELETE** `/api/v1/products/{id}` - [Deletar produto](/api-reference/api/v1/products/{id}/delete)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
