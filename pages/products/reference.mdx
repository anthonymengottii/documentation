## Produtos

Gerencie seu catálogo de produtos que podem ser associados a links de pagamento.

<Note>
  Para um guia completo sobre Produtos, consulte [Products Guide](/pages/guides/features/products).
</Note>

## Endpoints Disponíveis

- **POST** `/api/v1/products` - [Criar um novo produto](/pages/products/create)
- **GET** `/api/v1/products` - [Listar produtos](/pages/products/list)
- **GET** `/api/v1/products/{id}` - [Obter produto específico](/pages/products/get)
- **PATCH** `/api/v1/products/{id}` - [Atualizar produto](/pages/products/update)
- **DELETE** `/api/v1/products/{id}` - [Deletar produto](/pages/products/delete)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
