## Cupons

Crie e gerencie cupons de desconto para aplicar em pagamentos.

<Note>
  Para um guia completo sobre Cupons, consulte [Coupons Guide](/pages/guides/features/coupons).
</Note>

## Endpoints Disponíveis

- **POST** `/api/v1/coupons` - [Criar um novo cupom](/api-reference/api/v1/coupons/post)
- **GET** `/api/v1/coupons` - [Listar cupons](/api-reference/api/v1/coupons/get)
- **GET** `/api/v1/coupons/{id}` - [Obter cupom específico](/api-reference/api/v1/coupons/{id}/get)
- **PATCH** `/api/v1/coupons/{id}` - [Atualizar cupom](/api-reference/api/v1/coupons/{id}/patch)
- **DELETE** `/api/v1/coupons/{id}` - [Deletar cupom](/api-reference/api/v1/coupons/{id}/delete)
- **POST** `/api/v1/coupons/validate` - [Validar cupom (público)](/api-reference/api/v1/coupons/validate/post)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key), exceto `/api/v1/coupons/validate` que é público.

```
Authorization: Bearer SUA_API_KEY
```
