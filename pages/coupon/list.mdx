## Listar Cupons

Retorna uma lista paginada de todos os cupons criados pelo usuário.

<RequestExample>

```bash
curl -X GET "https://api.upay-sistema.onrender.com/api/coupons?page=1&limit=20" \
  -H "Authorization: Bearer SUA_API_KEY"
```

</RequestExample>

<ParamField query="page" type="number">
  Número da página (padrão: 1)
</ParamField>
<ParamField query="limit" type="number">
  Itens por página (padrão: 20, máx: 100)
</ParamField>
<ParamField query="status" type="enum">
  Filtrar por status: "ACTIVE" ou "INACTIVE"
</ParamField>

<ResponseField name="success" type="boolean">
  Indica se a requisição foi bem-sucedida
</ResponseField>
<ResponseField name="data" type="array">
  Lista de cupons
</ResponseField>
<ResponseField name="pagination" type="object">
  Informações de paginação
</ResponseField>
