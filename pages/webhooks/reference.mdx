## Webhooks

Configure webhooks para receber notificações em tempo real sobre eventos importantes.

<Note>
  Para um guia completo sobre Webhooks, consulte [Webhooks Guide](/pages/guides/features/webhooks).
</Note>

## Endpoints Disponíveis

- **GET** `/api/v1/webhooks` - [Obter informações de webhook](/pages/webhooks/get)
- **GET** `/api/v1/webhooks/subscriptions` - [Listar assinaturas de webhook](/pages/webhooks/subscriptions-list)
- **POST** `/api/v1/webhooks/subscriptions` - [Criar nova assinatura](/pages/webhooks/subscriptions-create)
- **PUT** `/api/v1/webhooks/subscriptions/{id}` - [Atualizar assinatura](/pages/webhooks/subscriptions-update)
- **DELETE** `/api/v1/webhooks/subscriptions/{id}` - [Deletar assinatura](/pages/webhooks/subscriptions-delete)
- **GET** `/api/v1/webhooks/events` - [Listar eventos de webhook](/pages/webhooks/events)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
