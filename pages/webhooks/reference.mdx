## Webhooks

Configure webhooks para receber notificações em tempo real sobre eventos importantes.

<Note>
  Para um guia completo sobre Webhooks, consulte [Webhooks Guide](/pages/guides/features/webhooks).
</Note>

## Endpoints Disponíveis

- **GET** `/api/v1/webhooks` - [Obter informações de webhook](/api-reference/api/v1/webhooks/get)
- **GET** `/api/v1/webhooks/subscriptions` - [Listar assinaturas de webhook](/api-reference/api/v1/webhooks/subscriptions/get)
- **POST** `/api/v1/webhooks/subscriptions` - [Criar nova assinatura](/api-reference/api/v1/webhooks/subscriptions/post)
- **PUT** `/api/v1/webhooks/subscriptions/{id}` - [Atualizar assinatura](/api-reference/api/v1/webhooks/subscriptions/{id}/put)
- **DELETE** `/api/v1/webhooks/subscriptions/{id}` - [Deletar assinatura](/api-reference/api/v1/webhooks/subscriptions/{id}/delete)
- **GET** `/api/v1/webhooks/events` - [Listar eventos de webhook](/api-reference/api/v1/webhooks/events/get)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
