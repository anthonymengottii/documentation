## Clientes

Gerencie seus clientes e suas informações cadastrais.

<Note>
  Para um guia completo sobre Clientes, consulte [Customers Guide](/pages/guides/features/customers).
</Note>

## Endpoints Disponíveis

- **POST** `/api/v1/clients` - [Criar um novo cliente](/pages/customers/create)
- **GET** `/api/v1/clients` - [Listar clientes](/pages/customers/list)
- **GET** `/api/v1/clients/{id}` - [Obter cliente específico](/pages/customers/get)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
