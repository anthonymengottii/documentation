## Links de Pagamento

Gerencie links de pagamento personalizados que podem ser compartilhados com seus clientes.

<Note>
  Para um guia completo sobre Links de Pagamento, consulte [Guia de Links de Pagamento](/pages/guides/features/payment-links).
</Note>

## Endpoints Disponíveis

- **POST** `/api/v1/payment-links` - [Criar um novo link de pagamento](/pages/payment-links/create)
- **GET** `/api/v1/payment-links` - [Listar links de pagamento](/pages/payment-links/list)
- **GET** `/api/v1/payment-links/{id}` - [Obter link de pagamento específico](/pages/payment-links/get)
- **PATCH** `/api/v1/payment-links/{id}` - [Atualizar link de pagamento](/pages/payment-links/update)
- **DELETE** `/api/v1/payment-links/{id}` - [Deletar link de pagamento](/pages/payment-links/delete)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
