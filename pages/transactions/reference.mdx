## Transações

Consulte e gerencie transações de pagamento.

## Endpoints Disponíveis

- **POST** `/api/v1/transactions` - [Criar uma nova transação](/api-reference/api/v1/transactions/post)
- **GET** `/api/v1/transactions` - [Listar transações](/api-reference/api/v1/transactions/get)
- **GET** `/api/v1/transactions/{id}` - [Obter transação específica](/api-reference/api/v1/transactions/{id}/get)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```

## Base URL

```
https://api.upay-sistema.onrender.com
```
