## Clientes

Gerencie a base de clientes da sua loja. Os clientes podem ser associados a transações e links de pagamento.

<Note>
  Para integração rápida, utilize nossos [SDKs oficiais](/pages/sdks) disponíveis em JavaScript, Python, PHP e Java.
</Note>

## Endpoints Disponíveis

- **POST** `/api/v1/clients` - [Criar um novo cliente](/pages/clients/create)
- **GET** `/api/v1/clients` - [Listar clientes](/pages/clients/list)
- **GET** `/api/v1/clients/{id}` - [Obter cliente específico](/pages/clients/get)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key) com as seguintes permissões:

- `clients:read` - Para listar e obter clientes
- `clients:write` - Para criar clientes

```
Authorization: Bearer SUA_API_KEY
```

## Modelo de Dados

### Cliente

```json
{
  "id": "uuid",
  "name": "João Silva",
  "email": "joao@example.com",
  "document": "12345678900",
  "phone": "+5511999999999",
  "createdAt": "2025-01-15T10:30:00Z",
  "updatedAt": "2025-01-15T10:30:00Z"
}
```

### Campos

| Campo | Tipo | Descrição | Obrigatório |
|-------|------|-----------|-------------|
| `id` | string (UUID) | Identificador único do cliente | Sim |
| `name` | string | Nome completo do cliente | Sim |
| `email` | string | Email do cliente (único por usuário) | Sim |
| `document` | string | CPF/CNPJ do cliente | Não |
| `phone` | string | Telefone do cliente | Não |
| `createdAt` | datetime | Data de criação | Sim |
| `updatedAt` | datetime | Data da última atualização | Sim |
