## Criar Cliente

Cria um novo cliente na base de dados do usuário autenticado pela API Key.

<Warning>
  Requer permissão `clients:write` na API Key. O email deve ser único por usuário.
</Warning>

## Corpo da Requisição

| Campo | Tipo | Descrição | Obrigatório | Validação |
|-------|------|-----------|-------------|-----------|
| `name` | string | Nome completo do cliente | Sim | Mínimo 3 caracteres, máximo 255 |
| `email` | string | Email do cliente | Sim | Email válido, único por usuário |
| `document` | string | CPF/CNPJ do cliente | Não | Mínimo 11 caracteres, máximo 20 |
| `phone` | string | Telefone do cliente | Não | Mínimo 10 caracteres, máximo 20 |

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X POST "https://upay-sistema-api.onrender.com/api/v1/clients" \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "João Silva",
    "email": "joao@example.com",
    "document": "12345678900",
    "phone": "+5511999999999"
  }'
```

```javascript JavaScript SDK
const client = await upay.clients.create({
  name: 'João Silva',
  email: 'joao@example.com',
  document: '12345678900',
  phone: '+5511999999999'
});

console.log(client);
```

```python Python SDK
client = upay.clients.create({
    'name': 'João Silva',
    'email': 'joao@example.com',
    'document': '12345678900',
    'phone': '+5511999999999'
})

print(client)
```

```php PHP SDK
$client = $upay->clients->create([
    'name' => 'João Silva',
    'email' => 'joao@example.com',
    'document' => '12345678900',
    'phone' => '+5511999999999'
]);

print_r($client);
```

</CodeGroup>

## Validações

### Email Duplicado

Se um cliente com o mesmo email já existe para o usuário, a API retornará:

```json
{
  "message": "Um cliente com este e-mail já existe."
}
```

**Status Code:** `409 Conflict`

## Códigos de Status

| Código | Descrição |
|--------|-----------|
| `201` | Cliente criado com sucesso |
| `400` | Dados inválidos (validação falhou) |
| `401` | Não autenticado (API Key inválida) |
| `403` | Sem permissão (`clients:write`) |
| `409` | Email já existe para este usuário |
