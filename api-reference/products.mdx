## Produtos

Gerencie seu catálogo de produtos que podem ser associados a links de pagamento.

<Note>
  Para um guia completo sobre Produtos, consulte [Products Guide](/guides/features/products).
</Note>

## Listar Produtos

<RequestExample>

```bash
curl -X GET "https://upay-sistema-api.onrender.com/api/products?page=1&limit=20" \
  -H "Authorization: Bearer SUA_API_KEY"
```

</RequestExample>

<ResponseField name="success" type="boolean">
  Indica se a requisição foi bem-sucedida
</ResponseField>
<ResponseField name="data" type="array">
  Lista de produtos
</ResponseField>
<ResponseField name="pagination" type="object">
  Informações de paginação
</ResponseField>

## Criar Produto

<RequestExample>

```bash
curl -X POST "https://upay-sistema-api.onrender.com/api/products" \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Curso de JavaScript",
    "description": "Curso completo de JavaScript",
    "price": 19900,
    "stockQuantity": 100,
    "stockEnabled": true,
    "isDigital": false,
    "status": "ACTIVE"
  }'
```

</RequestExample>

<ParamField body="name" type="string" required>
  Nome do produto (máx. 255 caracteres)
</ParamField>
<ParamField body="description" type="string">
  Descrição detalhada do produto
</ParamField>
<ParamField body="price" type="number" required>
  Preço em centavos (ex: 19900 = R$ 199,00). Deve ser inteiro positivo.
</ParamField>
<ParamField body="stockQuantity" type="number">
  Quantidade em estoque. Obrigatório quando `stockEnabled` for true.
</ParamField>
<ParamField body="stockEnabled" type="boolean">
  Ativar controle de estoque (padrão: false)
</ParamField>
<ParamField body="isDigital" type="boolean">
  Indica se o produto é digital (sem entrega física). Padrão: false.
</ParamField>
<ParamField body="imageUrl" type="string">
  URL da imagem do produto. Deve ser uma URL válida.
</ParamField>
<ParamField body="status" type="enum">
  "ACTIVE" ou "INACTIVE" (padrão: "ACTIVE")
</ParamField>

## Obter Produto

<RequestExample>

```bash
curl -X GET "https://upay-sistema-api.onrender.com/api/products/{id}" \
  -H "Authorization: Bearer SUA_API_KEY"
```

</RequestExample>

## Atualizar Produto

<RequestExample>

```bash
curl -X PATCH "https://upay-sistema-api.onrender.com/api/products/{id}" \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "price": 17900
  }'
```

</RequestExample>

## Atualizar Estoque

<RequestExample>

```bash
curl -X PATCH "https://upay-sistema-api.onrender.com/api/products/{id}/stock" \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "stockQuantity": 150
  }'
```

</RequestExample>

## Deletar Produto

<RequestExample>

```bash
curl -X DELETE "https://upay-sistema-api.onrender.com/api/products/{id}" \
  -H "Authorization: Bearer SUA_API_KEY"
```

</RequestExample>
