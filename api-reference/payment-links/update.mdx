## Atualizar Link de Pagamento

Atualiza um ou mais campos de um link de pagamento existente.

<RequestExample>

```bash
curl -X PATCH "https://api.upay-sistema.onrender.com/api/payment-links/{id}" \
  -H "Authorization: Bearer SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Novo Título",
    "status": "INACTIVE"
  }'
```

</RequestExample>

<ParamField path="id" type="string" required>
  ID único do link de pagamento
</ParamField>
<ParamField body="title" type="string">
  Novo título (opcional)
</ParamField>
<ParamField body="description" type="string">
  Nova descrição (opcional)
</ParamField>
<ParamField body="amountCents" type="number">
  Novo valor em centavos (opcional)
</ParamField>
<ParamField body="status" type="enum">
  Novo status: "ACTIVE" ou "INACTIVE" (opcional)
</ParamField>

<Note>
  Você pode atualizar apenas os campos que desejar. Campos não enviados permanecerão inalterados.
</Note>

<ResponseField name="success" type="boolean">
  Indica se a requisição foi bem-sucedida
</ResponseField>
<ResponseField name="data" type="object">
  Dados atualizados do link de pagamento
</ResponseField>
