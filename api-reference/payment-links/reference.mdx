## Payment Links

Gerencie links de pagamento personalizados que podem ser compartilhados com seus clientes.

<Note>
  Para um guia completo sobre Payment Links, consulte [Payment Links Guide](/pages/guides/features/payment-links).
</Note>

## Endpoints Disponíveis

- **POST** `/api/payment-links` - Criar um novo link de pagamento
- **GET** `/api/payment-links` - Listar links de pagamento
- **GET** `/api/payment-links/:id` - Obter link de pagamento específico
- **PATCH** `/api/payment-links/:id` - Atualizar link de pagamento
- **DELETE** `/api/payment-links/:id` - Deletar link de pagamento

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
