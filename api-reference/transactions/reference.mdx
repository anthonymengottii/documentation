## Transações

Consulte e gerencie transações de pagamento.

## Endpoints Disponíveis

- **POST** `/api/v1/transactions` - Criar uma nova transação
- **GET** `/api/v1/transactions` - Listar transações
- **GET** `/api/v1/transactions/:id` - Obter transação específica

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key).

```
Authorization: Bearer SUA_API_KEY
```
