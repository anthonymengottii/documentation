## Cupons

Crie e gerencie cupons de desconto para aplicar em pagamentos.

<Note>
  Para um guia completo sobre Cupons, consulte [Coupons Guide](/pages/guides/features/coupons).
</Note>

## Endpoints Disponíveis

- **POST** `/api/coupons` - Criar um novo cupom
- **GET** `/api/coupons` - Listar cupons
- **GET** `/api/coupons/:id` - Obter cupom específico
- **PATCH** `/api/coupons/:id` - Atualizar cupom
- **DELETE** `/api/coupons/:id` - Deletar cupom
- **POST** `/api/coupons/validate` - Validar cupom (público)

## Autenticação

Todos os endpoints requerem autenticação via Bearer Token (API Key), exceto `/api/coupons/validate` que é público.

```
Authorization: Bearer SUA_API_KEY
```
